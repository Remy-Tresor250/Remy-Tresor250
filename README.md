## 🐝 Founder & Maintainer — Bee (Open-Source AI Knowledge Hive)

I founded and maintain **[Bee](https://github.com/bee-ai-labs/bee)** — *the open-source AI knowledge hive*: a community-driven library of modern AI engineering knowledge with runnable examples, clear theory, and guided learning paths. Think **MDN for AI Engineering**.

[![Bee Repo](https://img.shields.io/badge/GitHub-bee--ai--labs%2Fbee-F5A623?style=for-the-badge&logo=github&logoColor=white)](https://github.com/bee-ai-labs/bee) [![Docs](https://img.shields.io/badge/Docs-bee.atomiq.rw-FF7A00?style=for-the-badge&logo=readthedocs&logoColor=white)](https://bee.atomiq.rw) [![Stars](https://img.shields.io/github/stars/bee-ai-labs/bee?style=for-the-badge&logo=github&color=FFD54A&logoColor=black)](https://github.com/bee-ai-labs/bee/stargazers) [![Contribute](https://img.shields.io/badge/Contribute-Good_First_Issues-2EA043?style=for-the-badge&logo=github&logoColor=white)](https://github.com/bee-ai-labs/bee/labels/good%20first%20issue)

**What Bee covers:** LLMs & transformers · Prompt engineering · RAG · AI agents (MCP, multi-agent) · Vision & speech · Evaluation, security & deployment.

**How I run the community:**
- 🧭 **Direction & roadmap** — I set the vision, quality bar (*everything runs, no placeholders*), and public roadmap.
- 🔍 **Reviews & mentorship** — I review pull requests and help first-time contributors land their first merge.
- 🤝 **Contributor-first** — issue/PR templates, `good first issue` labels, learning paths, and a welcoming Code of Conduct so anyone can contribute.
- ⚙️ **Maintainer engineering** — CI/CD (lint, spell-check, link-check, docs build), tested examples, and a MkDocs documentation site.

> 🌍 Read the docs at **[bee.atomiq.rw](https://bee.atomiq.rw)** · ⭐ Star the [repo](https://github.com/bee-ai-labs/bee) to support the hive.
